# University level cover images

Drop the three branded thumbnails here using these exact filenames so
the `LevelCollectionCard` components on the University page pick them
up automatically:

| File                       | Card        | Brief                                                  |
| -------------------------- | ----------- | ------------------------------------------------------ |
| `level-beginner.jpg`       | Beginner    | "STARTING THE JOURNEY" — notebook / books / FOCUS.     |
| `level-intermediate.jpg`   | Intermediate| "BUILDING SKILLS & GROWING" — laptop / LEVEL UP.       |
| `level-advanced.jpg`       | Advanced    | "TEACHING & LEADING OTHERS" — audience / LEADERSHIP.   |

Recommended:
- Format: JPEG, ~1600px wide max (re-compressed if larger).
- Aspect: roughly 16:10 — cards crop with `background-size: cover`
  centered, so anything close to landscape works.
- Same dark aesthetic across all three so the row reads as a set.

The `Topics` card (level 4) intentionally has no cover — it keeps the
icon-tint treatment as the off-path bucket.
